Rupantaran Documentation
=======================

Welcome to Rupantaran's documentation. Rupantaran is a Python package that converts various Nepali-specific measurements into SI units or commonly used metric units.

Quick Start
----------

.. code-block:: bash

   pip install rupantaran


Installation
===========

Requirements
-----------
- Python 3.7+

Installation Steps
----------------

1. Using pip:
   
   .. code-block:: bash

      pip install rupantaran

2. From source:

   .. code-block:: bash

      git clone https://github.com/biraj094/rupantaran
      cd rupantaran
      pip install -e .
